# Phase C - Cross-Lab Reproducibility

## Objective

Establish independent reproducibility across multiple labs and operator teams.

## Setup

- at least three independent sites
- replicated hardware bill of materials and calibration protocol
- central blinded coordinator for trial token generation

## Required Controls

- site-specific null runs
- instrumentation audit per site
- standardized artifact ledger format

## Metrics

- per-site balanced accuracy and BER
- inter-site variance of core metrics
- pooled reproducibility confidence interval

## Pass Gate

- all sites meet minimum thresholds
- pooled results exceed reproducibility target
- no unresolved confounds in audit trail
