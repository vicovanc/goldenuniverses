# Phase A - Benchtop Validation

## Objective

Demonstrate controlled detectability and lock stability in a tightly
instrumented benchtop setup.

## Setup

- short-distance TX/RX bench
- full shielding and dummy-channel controls
- no environmental mobility

## Test Blocks

1. baseline null block
2. assisted activation block
3. active activation block
4. repeated lock/relock stress block

## Primary Metrics

- SNR distribution by block
- lock acquisition time
- BER on known pilot/payload sequence
- null false-positive rate

## Pass Gate

- active block significantly above null baseline
- stable lock below target time
- no unresolved leakage artifact
