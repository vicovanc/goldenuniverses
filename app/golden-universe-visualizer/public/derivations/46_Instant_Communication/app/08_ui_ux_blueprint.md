# 08 UI/UX Blueprint

## Visual direction

Bluesteel clean desktop style: minimal chrome, high information density, explicit state.

## Primary screens

- Mission Control: run setup, state machine, operator actions
- Live Telemetry: sensor traces, drift, quality warnings
- Decode Console: blinded trial stream and confidence
- Benchmark Board: threshold bars and gate verdict
- Evidence Vault: artifacts, hashes, chain-of-custody

## Interaction principles

- no hidden configuration updates
- destructive actions require typed confirmation
- every state transition visibly logged
