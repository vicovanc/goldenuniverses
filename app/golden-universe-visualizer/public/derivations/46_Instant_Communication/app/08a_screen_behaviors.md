# 08a Screen Behaviors

## Mission Control

- run create/preregister/start/stop controls
- explicit state timeline and lock status
- hard-stop alerts with required acknowledgments

## Live Telemetry

- multi-channel synchronized plots
- quality bars (drop/jitter/drift)
- envelope breach markers

## Decode Console

- blinded decode stream with confidence
- dual-pipeline consistency indicator
- reveal controls disabled until seal complete

## Benchmark Board

- threshold cards with pass/fail status
- all-gates summary and promotion status
- trace links to run artifacts

## Evidence Vault

- hash chain table
- artifact tree by provenance
- export package for independent audit
