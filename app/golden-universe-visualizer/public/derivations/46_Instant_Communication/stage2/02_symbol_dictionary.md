# 02 Symbol Dictionary

| Symbol | Meaning | Units |
|---|---|---|
| `rho` | field amplitude proxy | arb |
| `theta` | phase channel variable | rad |
| `R` | memory state | arb |
| `beta` | memory decay constant | s^-1 |
| `kappa` | topological coupling level | dimensionless |
| `u_tx` | TX actuation signal | arb |
| `y_rx` | RX observable | arb |
| `g_hat` | estimated channel gain | dimensionless |
| `e` | control error | arb |
| `n_rx` | receiver noise | arb |
| `SNR_db` | signal-to-noise ratio | dB |
| `BER` | bit error rate | probability |
| `t_lock` | lock time | s |

All symbols are required in Stage2 reports and interface documents.
