# 11 Fabrication and Assembly Spec

## Build Sequence

1. Prepare substrate and verify flatness
2. Deposit/attach active material stack
3. Integrate TX actuation electrode set
4. Integrate RX sensing front-end
5. Seal and mount environmental shielding
6. Perform continuity and noise-floor prechecks

## Required Checks

- visual metrology and alignment report
- baseline noise-floor characterization
- transfer-path continuity test
- calibration pre-run acceptance

## Assembly Rejection Criteria

- thermal instability above allowed envelope
- unresolved connector intermittency
- excessive baseline drift in null mode
