# 08 Material Selection Spec

## Candidate Classes

1. Pi-stacked molecular arrays
2. Phase-phonon crystal substrates
3. Hybrid bioelectric interface stacks

## Scoring Axes

- channel coupling strength
- coherence persistence time
- manufacturability maturity
- instrumentation compatibility

## Baseline Selection (Lab Scale)

Primary: pi-stacked arrays on low-vibration crystal substrate.

## Tolerances

- substrate flatness: <= 2 um across active region
- thermal drift: <= 0.1 C/hour during active runs
- assembly misalignment: <= 50 um TX-RX fixture reference
